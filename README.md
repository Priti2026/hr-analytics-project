# hr-analytics-project
- HR Analytics project using Power Bi, SQL and Python
# Objective
- To analyze employees attrition in the company.
# Tools Used
- SQL
- Power BI
- Python (for data cleaning)
# Dataset Information
- The dataset contains employees details such as 
   department, gender, education field, job role, salary, and attrition status.
# Steps
- Cleaned the dataset
- Calculated attrition rate
- Analyzed attrition by gender and department
- Created dashboard in Power BI
# Key Insights
- High attrition in Life Sciences
- Low salary employees had higher attrition
# Conclusion
This project helped in understanding employees turnover patterns and how data visualization can support HR decision-making.
